// 起動時にGoogleスプレッドシートから名簿を自動取得 async function loadSpreadsheetData() { // ★修正ポイント：「ウェブに公開」を使わず、通常の共有リンクのままJSON形式のデータを取得するGoogle公式APIのルートに変更しました const url = `https://google.com{SPREADSHEET_ID}/gviz/tq?tqx=out:json`; try { const response = await fetch(url); const rawText = await response.text(); // Google API特有のノイズテキスト（google.visualization...）を除去して純粋なJSONにする const jsonText = rawText.substring(rawText.indexOf('{'), rawText.lastIndexOf('}') + 1); const dataObj = JSON.parse(jsonText); const rows = dataObj.table.rows; tournamentData = {}; // スプレッドシートの行をループ処理 rows.forEach((row, index) => { const cells = row.c; if (cells && cells[0] && cells[0].v) { const teamName = cells[0].v; const members = []; // 選手1〜4の名前を取得（空なら空文字を入れる） for (let i = 1; i <= 4; i++) { members.push((cells[i] && cells[i].v) ? cells[i].v.toString() : ""); } tournamentData["team_" + (index + 1)] = { name: teamName, members: members }; } }); // 左右のプルダウンの選択肢を生成 ['leftTeamSelect', 'rightTeamSelect'].forEach(id => { let h = ''; for(let k in tournamentData) h += ``; document.getElementById(id).innerHTML = h; }); document.getElementById('loadStatus').innerText = "🟢 スプレッドシート名簿と正常に同期しました"; document.getElementById('loadStatus').style.color = "#00ffcc"; } catch (e) { console.error(e); document.getElementById('loadStatus').innerText = "❌ 名簿の同期に失敗しました（インターネット接続や共有設定を確認してください）"; document.getElementById('loadStatus').style.color = "#ff3c12"; } }
